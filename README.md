Having a trouble to understand something like quick sort? Try functional language like Haskell. 

**The implementation is so obvious just the description of the algorithm.**
 ```haskell
 qsort :: Ord a => [a] -> [a]
 qsort []       = []
 qsort (x:xs)   = [x' | x'<- xs, x' <= x ] ++ [x] ++ 
                  [x' | x'<- xs, x' > x ]
 ```
